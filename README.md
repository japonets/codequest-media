<h2>📌 Sobre</h2>
<p>
  Este repositório é destinado à organização dos conteúdos audiovisuais
  do CODEQUEST.
</p>

<p>
  Aqui serão reunidos roteiros, planejamentos de gravação, materiais
  relacionados à edição e estruturação dos conteúdos em vídeo do projeto.
</p>

---

<h2>🎯 Objetivo</h2>
<ul>
  <li>Organizar roteiros e conteúdos para gravação</li>
  <li>Planejar e acompanhar a produção de vídeos</li>
  <li>Centralizar materiais de edição</li>
  <li>Padronizar os processos audiovisuais do projeto</li>
</ul>
